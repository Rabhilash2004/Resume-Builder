🚀 AI Resume Builder Web App

A modern and user-friendly Resume Builder Web Application that allows users to create professional resumes quickly with real-time preview and easy customization.

✨ Features
🔐 User Authentication (Login / Signup)
🧾 Dynamic Resume Creation
👀 Live Preview (real-time updates while typing)
➕ Add/Edit Sections:
Personal Details
Skills
Experience
Education
Projects
Certifications
📊 Resume Completion Tracker
🎨 Customizable Styles (font size, accent color)
📄 Download Resume as PDF
⚡ Smooth and responsive UI
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Concepts Used: DOM Manipulation, Event Handling, Local Storage
Optional Backend: Can be extended using Java (Spring Boot) / Node.js

🎯 Project Objective

The goal of this project is to simplify resume creation by providing a clean, intuitive interface where users can build professional resumes without technical knowledge.
